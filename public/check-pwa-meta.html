PWA Meta Tag Checker

Instructions

This tool checks if your site has the necessary PWA meta tags for iOS standalone mode.

To use this on iOS:

  1. Add this page to your home screen
  2. Open the app from your home screen
  3. Check if it runs in standalone mode (no Safari UI)

Required JS Code

// Add this to your app's JavaScript to properly detect standalone mode: if (window.navigator.standalone === true) { console.log("App is running in standalone mode (iOS)"); // You can add a class to the body for PWA-specific styling document.body.classList.add('pwa-standalone'); }
